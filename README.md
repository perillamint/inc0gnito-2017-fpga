# Inc0gnito CTF 2017 - Verilog Edition

## Note
이 문제의 출력은, 난이도 조정을 위해 `INC0{.*}` 포멧을 따르지 않습니다.<br>
문제의 결과로 얻어낸 4-byte Null-terminated 키를 `INC0{}` 안에 넣어 제출하십시오.

## 키 조건
* 키는 Null-terminated string 입니다. (예: 41424300 => ABC)
* 엔디안은 리틀 엔디안입니다.
* 0x00000000 은 키가 아닙니다.
